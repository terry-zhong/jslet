Jslet is an data-sensitive javascript framework. You can use it to develop powerful web application quickly with less code.

Features

  • Load-on-Demand
Jslet is base on an excellent architecture, so its source code is small. Nevertheless, we provide the "Load-on-Demand" feature.
  • Base on Data Model
Just only define data models and their relations, and Jslet can do the remain things for you. It's a very specific feature, and it is the reason why you write less code with Jslet.
  • Powerful Local Data Manipulation

User can insert/update/delete data, filter data, find data, sort data, format data and validate data in client side. There are some specific features: master-detail style data, context rule, field chain etc.

  • Data-Sensitive UI Controls
In Jslet, there are 30+ Data-Sensitive UI controls in Jslet which cover almost web application requirement. Here we strongly recommend these controls: DBTable, DBTreeView, DBChart, DBEditPanel etc, these are really powerful, you must try them.
  • Comprehensive Data Validation
Three level data validations:
  1. Key pressing valiation: check data according field data type;
  2. Field validation: check data according regular expression or custom validation;
  3. Record validation: check data after finishing editing a record.
  • Perfect Performance
Perfect performance in loading data, refreshing data and applying changes to server; In other frameworks, large data is a big problem, but in jslet you need not warry about this issue. You can load unlimited data into DBTable and DBTreeView control.
  • Simple Interaction with Server
Only three methods to connect to server: applyQuery, applyChanges and applySelected, you do not care about how to combine data and how to refresh UI controls, all you need to provide is URL.
  • Rich Configuration
You can configurate dataset and UI controls, like field label, display format, default value, filter expression, validation rule, editor layout and so on.
So, you can build more flexible UI page with Jslet.
  • Support I18N, UI Theme, Multiple-Browser
Support I18N, especially support middle east languages;
Support UI theme, and you can define your UI theme;
Support all modern browsers, like IE 7+, Chrome, Firefox, Safari and so on;
  • Easy and Simple Extension
You can define your UI controls easily.
