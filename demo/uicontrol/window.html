Jslet - Window Demo

Demonstrate how to use Window control.

Demo Content:

  1. Open modeless window in the "Body"(with icon)
  2. Open model window in the "Body"
  3. Open http://jslet.github.com/jslet' in window
  4. Demostate the features: minimize button, maximize button, double click window title to maximize, change window size, move window;
  5. Open modeless window in one "Div" (Can open many windows)
  6. Open model window in on "DIV"
  7. Oen window which content is one "Div" of this page
