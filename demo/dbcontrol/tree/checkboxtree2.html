Jslet - Selectable DBTreeView(Tree node is selectable or Not) Demo

Demonstrate how to configurate tree node has check box or not.
CheckBox Tree

Demo Content:

  1. After check a child node, all of its parent node will display bold, it is very useful to find all checked nodes when the parent nodes are collapsed.
  2. Check tree node separately. You can right click on a tree node to select/unselect all.
  3. Set which tree nodes have check box. In this sample, only leaf tree nodes have check box.
