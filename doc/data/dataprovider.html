      • query

        query(dataset, url, condition, pageNo, pageSize, successHandler, errorHandler)
        Retrieve data from server. This method is used by Dataset Object.
        Parameters:
        dataset - (Required) jslet.data.Dataset, dataset Object.
        url - (Required) String, query url.
        condition - (optional) JSON object, query parameter.
        pageNo - (Required) Integer, Page number.
        pageSize - (Required) Integer, Page size.
        successHandler - (Required) Function, call back function for successful query.
        errorHandler - (Required) Function, call back function for failure query.

      • submitSelected

        submitSelected(dataset, url, selectedData, successHandler, errorHandler, deleteOnSuccess)
        Send selected data to server whether or not the records have been changed.
        This method is used by Dataset Object.
        Parameters:
        dataset - (Required) jslet.data.Dataset, dataset Object.
        url - (Required) String, submit url.
        selectedData - (optional) Array of JSON object, selected records.
        successHandler - (Required) Function, call back function for successful submit.
        errorHandler - (Required) Function, call back function for failure submit.
        deleteOnSuccess - (optional) Boolean, identify delete the selected record or not if processing successfully at server side.
        Since:
        1.0
