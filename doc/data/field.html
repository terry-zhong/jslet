jslet.data.

      • createField

        jslet.data.createField(fieldConfig, parent)
        Create a field object. Example: 
        
var fldObj = jslet.data.createField({name:'field1', type:'N', label: 'field1 label'});
        Parameters:
        fieldConfig - (Required) Object, a json object which property names are same as jslet.data.Field. Example: {name: 'xx', type: 'N', ...}.
        parent - (Optional) jslet.data.Field, the parent field of this field.
        Returns:
        Field Object.
        Since:
        1.0

      • createStringField

        jslet.data.createStringField(fldName, length, parent)

        Parameters:
