      • EditMask

        jslet.data.EditMask(mask, keepChar, transform)
        Constructs a new edit mask oject. Example: 
        
var maskObj = new jslet.data.EditMask('AA-0000', false, 'upper');
        Parameters:
        mask - (Required) String, mask String. 
        Rule:
'#': char set: 0-9 and -, not required
'0': char set: 0-9, required
'9': char set: 0-9, not required
'L': char set: A-Z,a-z, required
'l': char set: A-Z,a-z, not required
'A': char set: 0-9,a-z,A-Z, required
'a': char set: 0-9,a-z,A-Z, not required
'C': char set: any char, required
'c': char set: any char, not required
        keepChar - (Optional) Boolean, keep non-mask character or not. Example: 
        	Suppose mask string: AA-000
	if keepChar is true, result will be: CC-123
	if KeepChar is false, result will be: CC123
        transform - (Optional) String, transform input character into UpperCase or LowerCase, optional value: upper, lower or null.
        Returns:
        Edit mask object.
        Since:
        2.0
