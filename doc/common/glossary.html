Jslet-Glossary

Data-aware controls
If data changed, the UI controls will changed synchronously.
Lookup Field
It means a field value comes from another dataset.
Field chain
Field chain is combined with relative fields and separated with dot(.),form: 
field1.field2.field3....
Tips: all fields in the field chain except the last one must be lookup field. Example: 
dsEmployee.setFilter("[deptId.location] == 'NewYork'");
Dataset Field:
Dataset field is a virtual field, it points to another dataset.Generally, we use it in master-detail case.
Group Field
Group field is a virtual field, we use this field to group other fields. Group field can contain normal field and group field.
Context Rule
Context rule is the rule between fields, means some field's state will changed synchronously if one field's value changed.
