    • jslet.like

      jslet.like(testValue, pattern, escapeChar)
      Test if the specified value matchs the pattern, the format of pattern is same as SQL like. Sample: 
      
jslet.like('abc','%b%');// true
jslet.like('abc','%b'); // false
jslet.like('abc','a_c');// true
      Parameters:
      testValue - (Required) String, test value.
      pattern - (Required) String, test pattern, format same as SQL like.
      escapeChar - (Optional) String, use this character to escape special character, default is "\".
      Returns:
      Boolean, true for matched, false for otherwise.
      Since:
      1.0

    • jslet.between

      jslet.between(testValue, minValue, maxValue)
      Test if the specified value is between minValue and maxValue. Sample: 
      
jslet.between(4,2,5);// true
jslet.between('c','a','b'); // false
      Parameters:
      testValue - (Required) Object, test value.
      minValue - (Required) Object, minimum value.
      maxValue - (Required) Object, maximum value.
      Returns:
      Boolean, true for matched, false for otherwise.
      Since:
      1.0
