Jslet-Expression Introduction

Jslet is a flexible framework, you can use expression many places:

  1. Data filter expression;
  2. Data finding expression;
  3. Formula Expression;
  4. Property: displayFields of lookup field;
  5. Property: displayFields of DBTreeView control;

So what's Jslet expression? Actually, Jslet expression is javascript expression, only one difference is indicating field name.

In Jslet expression, we use "[]" to surround Jslet dataset name and field name. The rule is:

[datasetName!fieldName.subFieldName]
The part "datasetName" and "subFieldName" can be omitted. Sample:
  1. Normal: 
    [age]>25, [birthday].getMonth()==0;
  2. With dataset name: 
    [department!name] == 'Sales';
  3. With field chain: 
    [department.location] == 'NewYork';
  4. Using jslet global functions: 
    
jslet.like([empName], 'T%') 
    
jslet.between([age], 20, 30)
    
jslet.inlist([age], 22, 25, 26)
