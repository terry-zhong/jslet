jslet.ui.

Class DBEditPanel;


  • DBEditPanel is used to input data, you can customize editor layout, like span columns, separator line: Sample: 
    
 var jsletParam = {type:"DBeditpanel",dataset:"employee", columnCount:3, columnWidth: 150, rowHeight:30, fields:[{field:"workerid",colSpan:2},{field:"department",colSpan:2,showLine:true}]};
 
 //1. Declaring:
      <div data-jslet='jsletParam' />
      
  //2. Binding
      <div id="ctrlId"  />
  	//Js snippet
 		var el = document.getElementById('ctrlId');
  	jslet.ui.bindControl(el, jsletParam);

  //3. Create dynamically
  	jslet.ui.createControl(jsletParam, document.body);
    You can also see Demo.

      columnGap Integer 10 The gap between label and editor.
      rowHeight Integer 30 Row height.
      onlySpecifiedFields Boolean false True - only show specified fields, false otherwise.
      fields Array of edit field configuration prototype: [{field: "field1", colSpan: 2}, ...].
