jslet.ui.

Class DBPageBar;


  • DBPageBar is used for pagination. Features:
    1. First page, Prior Page, Next Page, Last Page;
    2. Can go to specified page;
    3. Can specify page size on runtime;
    4. Need not write any code;
    Sample: 
    
 var jsletParam = {type:"DBPageBar",dataset:"bom",pageSizeList:[20,50,100,200]};
 
 //1. Declaring:
      <div data-jslet='jsletParam' />
      
  //2. Binding
      <div id="ctrlId"  />
  	//Js snippet
 		var el = document.getElementById('ctrlId');
  	jslet.ui.bindControl(el, jsletParam);

  //3. Create dynamically
  	jslet.ui.createControl(jsletParam, document.body);
    You can also see Demo.

    1.0

    • Properties

      Property Name Data Type Required Default Description
      type String Yes "DBPageBar" Must be: DBPageBar
      dataset String/jslet.data.Dataset Yes Dataset name or Dataset object.
      showPageSize Boolean true Identify if the "Page Size" part shows or not.
      showGotoButton Boolean true Identify if the "GoTo" part shows or not.
      pageSizeList Integer[] Page size list, like: [50,100,200].
