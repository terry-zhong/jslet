jslet.ui.

Class DBImage;


  • Display image which's url is from field value. Sample: 
    
 var jsletParam = {type:"DBImage",dataset:"employee",field:"photo"};
 
 //1. Declaring:
      <img data-jslet='type:"DBImage",dataset:"employee",field:"photo"' />
      or
      <img data-jslet='jsletParam' />
      
  //2. Binding
      <img id="ctrlId"  />
  	//Js snippet
 		var el = document.getElementById('ctrlId');
  	jslet.ui.bindControl(el, jsletParam);

  //3. Create dynamically
  	jslet.ui.createControl(jsletParam, document.body);
    You can also see Demo.

    1.0

    • Properties

      Property Name Data Type Required Default Description
      type String Yes "DBImage" Must be: DBImage
      dataset String/jslet.data.Dataset Yes Dataset name or Dataset object.
      field String Yes Field name
      baseUrl String The base url of image.
      altField String The field stores value for the image's alt.
      locked Boolean false True - DBImage does not refresh when user move the cursor to another record, false otherwise.
