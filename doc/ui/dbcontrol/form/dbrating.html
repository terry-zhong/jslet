jslet.ui.

Class DBRating;


  • A control which usually displays some star to user, and user can click to rate something. Sample: 
    
 var jsletParam = {type:"DBRating",dataset:"employee",field:"grade", itemCount: 5};
 
 //1. Declaring:
      <div data-jslet='jsletParam' />
      
  //2. Binding
      <div id="ctrlId"  />
  	//Js snippet
 		var el = document.getElementById('ctrlId');
  	jslet.ui.bindControl(el, jsletParam);

  //3. Create dynamically
  	jslet.ui.createControl(jsletParam, document.body);
    You can also see Demo.
    Since:
    1.0
    See Also:
    Jslet Global Functions

    • Properties

      Property Name Data Type Required Default Description
      type String Yes "DBRating" Must be: DBRating
      dataset String/jslet.data.Dataset Yes Dataset name or Dataset object.
      field String Yes Field name
      itemCount Integer 5 Rate item count, In other words, the count of 'Star' sign.
      splitCount Integer 0 You can use it to split the 'Star' sign to describe decimal like: 1.5, 1.25.
      itemWidth Integer 20 The width of one 'Star' sign.
      readOnly Boolean false read only or not.
      required Boolean false Required or not. If it is not required, you can rate 0 by double clicking first item.
