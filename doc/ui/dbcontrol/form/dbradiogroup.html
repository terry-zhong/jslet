jslet.ui.

Class DBRadioGroup;


  • DBRadioGroup displays a group of radio button. Sample: 
    
 var jsletParam = {type:"DBRadioGroup",dataset:"employee",field:"department", columnCount: 3};
 
 //1. Declaring:
      <div data-jslet='jsletParam' />
      
  //2. Binding
      <div id="ctrlId"  />
  	//Js snippet
 		var el = document.getElementById('ctrlId');
  	jslet.ui.bindControl(el, jsletParam);

  //3. Create dynamically
  	jslet.ui.createControl(jsletParam, document.body);
    You can also see Demo.
