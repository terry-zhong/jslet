jslet.ui.

Class DBAutoComplete


  • DBAutoComplete is a powerful control. Features:
    1. Delay displaying popup list after inputting some characters or elapsing short time.
    2. Searching field can be specified dynamically.
    Sample: 
    
 var jsletParam = {type:"DBAutoCompleteBox",field:"department", matchType:"start"};
 
 //1. Declaring:
      <input data-jslet='jsletParam' />
      
  //2. Binding
      <input id="ctrlId"  />
  	//Js snippet
 		var el = document.getElementById('ctrlId');
  	jslet.ui.bindControl(el, jsletParam);

  //3. Create dynamically
  	jslet.ui.createControl(jsletParam, document.body);
    You can also see Demo.

    1.0
