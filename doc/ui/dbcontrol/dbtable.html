h Jslet DBTable
jslet.ui.

Class DBTable;


  • DBTable is a powerful control. Below is its features:
    • Performance
      1. Perfect performance, you can load unlimited data into DBTable.
    • Appearance
      1. Can specify fixed columns or fixed rows;
      2. Show sequence column or not;
      3. Show select column or not, you can select a row or a group rows;
      4. Navigate data rows by clicking or cursor key(up, down, page up, page down);
      5. Can change column width by draging table header;
      6. Show total section or not.
      7. Show hyperlink on a field value or not;
      8. Can customize your table cell render;
      9. Show field value as a tip, it is very convenient for long field value.
      10. Merge same value of adjacent rows or not.
    • Stack Header
      1. Support stack header, and you can click stack header to sort data;
    • Sort by clicking table header
      1. Sort data by clicking table header, and toggle sorting direction by next clicking.
      2. Sort data with multiple field by 'CTRL' + clicking;
      3. If a field is a lookup field, it sorts data with the 'Code' field of lookup dataset, like: [deptId.code], because deptId is a sequence id, it isn't meanful;
    • Edit in cell
      • Edit data in cell, editable field is decided with jslet.data.Field: readOnly() and enabled()(), and edit control is specified jslet.data.Field: editControl();
    • Tree style table
      • Show data as a tree style table;
    Sample: 
    
 var jsletParam = {type:"DBTable",dataset:"employee"};
 
 //1. Declaring:
      <label data-jslet='jsletParam' />
      
  //2. Binding
      <label id="ctrlId"  />
  	//Js snippet
 		var el = document.getElementById('ctrlId');
  	jslet.ui.bindControl(el, jsletParam);

  //3. Create dynamically
  	jslet.ui.createControl(jsletParam, document.body);
    You can also see DBTable Demo.
