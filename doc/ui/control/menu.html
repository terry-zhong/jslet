jslet.ui.

Class MenuBar/Menu


  • There are two menu classes: jslet.ui.MenuBar and jslet.ui.Menu. You can use jslet.ui.MenuBar to create menu bar and use jslet.ui.Menu to create context menu or dropdown menu. Sample: 
    
 //There are 3 ways to create this control:
 var jsletParam = {type:"MenuBar", items:[{id:"file", name: "File"}, {id:"edit", name: "Edit"}]};
 //var jsletParam = {type:"Menu",  items:[{id:"new", name: "New"},{id:"open", name: "Open"}};
 //1. Declaring(recommend):
     <div data-jslet='jsletParam' style="width: 300px; height: 400px;"/>
      
  //2. Binding
  	<div id='ctrlId'>
  	</div>
  	//Js snippet
 		var el = document.getElementById('ctrlId');
  	jslet.ui.bindControl(el, jsletParam);
  		
  //3. Create dynamically
  	jslet.ui.createControl(jsletParam, document.body);
	
//Use the below code to show context menu
jslet('#ctrlId').showContextMenu(event);
//Show menu at the specified position
jslet('#ctrlId').show(left, top);
    You can also see Demo.

    • Properties

      Property Name Data Type Required Default Description
      type String Yes "MenuBar"/"Menu" Must be: "MenuBar"/"Menu"
      onItemClick Event Fired when user select a menu item. Event handler: 
      
function(menuId, checked) {...};
Parameters:
  menuId: String, menu item id.
  checked: Boolean, menu item is checked or not.
Return:
  void;
      items array of menu item

    • Menu Item Properties

      Property Name Data Type Required Default Description
      id String Menu item id
      name String Menu item name
      itemType String Menu item type, optional value: null, radio, check
      checked Boolean Menu item is checked or not, only work when itemType equals 'radio' or 'check'.
      group String Group name, only work when itemType equals 'radio'.
      onClick Event Fired when user select a menu item. Event handler: 
      
function(event) {...};
Parameters:
  event: mouse event.
Return:
  void;
      disabled Boolean false Menu item is disabled or not.
      iconClass String The icon css class name of menu item.
      disabledIconClass String The disabled icon css class name of menu item.
      items array of menu item

    • Methods

      Name Description
      show(left, top) Show menu at specified position.
      Parameters:
      left - (Required) Integer, Position left.
      top - (Required) Integer, Position top.
      Returns:
      void
      Since:
      2.0
      hide() Hide menu item and all its sub menu item.
      Since:
      2.0
      showContextMenu(event) Show menu on context menu. Example: 
      
<div id="popmenu" oncontextmenu="popMenu(event);" />
//js
function popMenu(event) {
jslet("#popmenu").showContextMenu(event);
}
      Parameters:
      event - (Required) Mouse event.
      Returns:
      void
      Since:
      2.0
      renderAll() Use the control configuration to render control in browser.
