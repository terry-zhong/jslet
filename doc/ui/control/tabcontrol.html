jslet.ui.

Class TabControl


  • jslet.ui.Window, it has the following features:
    1. User can add tab item dynamically;
    2. User can close tab item;
    3. Support context menu on tab item header;
    
 //There are 3 ways to create this control:
 var jsletParam = {type: "TabControl", 
		selectedIndex: 1, 
		onCreateContextMenu: doCreateContextMenu, 
		items: [
			{header: "one", userIFrame: true, url: "http://www.google.com", iconCls: "tabIcon"},
			{header: "two", closable: true, divId: "p2"},
			{header:"three",closable:true,divId:"p3"},
		]};
 //1. Declaring(recommend):
     <div data-jslet='jsletParam' style="width: 300px; height: 400px;"/>
      
  //2. Binding
  	<div id='ctrlId'>
  	</div>
  	//Js snippet
 		var el = document.getElementById('ctrlId');
  	jslet.ui.bindControl(el, jsletParam);
  		
  //3. Create dynamically
  	jslet.ui.createControl(jsletParam, document.body);
    You can also see Demo.
    Since:
    2.0
    See Also:

    • Properties

      Property Name Data Type Required Default Description
      type String Yes "TabControl" Must be: "TabControl"
      selectedIndex Integer 0 Selected tab item index
      newable Boolean true Allow user to add a new tab item or not.
      closable Boolean true Allow user to close current tab item or not.
      onAddTabItem Event Fired when user add a new tab item. Event handler: 
      
function() {...};
Return:
  jslet.ui.TabItem;
      onRemoveTabItem Event Fired when user delete a tab item. Event handler: 
      
function(tabIndex, selected) {...};
Parameters:
  tabIndex: Integer, the index of deleting tab item.
  selected: Boolean, the deleting tab item is selected or not.
Return:
  Boolean, true - can delete tab item, false otherwise;
      onSelectedChanged Event Fire when user toggle tab item. Event handler: 
      
function(oldIndex, newIndex) {...};
Parameters:
  oldIndex: Integer, old index.
  newIndex: Integer, new index.
Return:
  Boolean, true - can change tab item index, false otherwise;
      onCreateContextMenu Event Fire before show context menu, you can add your own context menu in this event. Example: 
      
tabObj.onCreateContextMenu = function(menuItems) {
  var menuObj = {id: "copy", name: "Copy", 
	onClick: doClick};
  menuItems.push(menuObj);
}
      Event handler: 
      
function(menuItems) {...};
Parameters:
  menuItems: Array of menu item.
Return:
  void;

    • jslet.ui.TabItem Properties

      Property Name Data Type Required Default Description
      id String Tab item id
      header String Tab item header
      iconClass String The icon css class name of menu item.
      itemType String Menu item type, optional value: null, radio, check
      closable Boolean false Identify user can close tab item or not.
      enable Boolean true Identify tab item is enabled or not.
      visible Boolean true Identify tab item is visible or not.
      content String Html content.
      useIFrame Boolean false Identify tab item is an iframe.
      url String The url of tab item.
