jslet.ui.

Class Window


  • jslet.ui.Window, it has the following features:
    1. Show as modal or modeless;
    2. User can change window size;
    3. User can minimize, maximize, restore or close window;
    4. User can move window;
    Example: 
    
    var oWin = jslet.ui.createControl({ type: "Window", iconcls:"winicon", caption: "test window", minimizable: false, maximizable:false,onActive: doWinActive, onPositionChanged: doPositionChanged });
    oWin.setContent("Open modeless window in the Body(with icon)!");
    oWin.show(350,250);
    //or oWin.showModel(350, 250);
    You can also see Demo.
    Since:
    2.0

    Jslet Global Functions

function(left, top) {...};

  top: Integer, top
Return:

      onActive Event Fired when user activates window. Event handler: 
      
function(self) {...};
Parameters:
  self: jslet.ui.Window, this window.
Return:
  void;

function(self) {...};
Parameters:
  self: jslet.ui.Window, this window.
Return:
  String, action name, optional value: close, hidden;
  if return value is hidden, window is hidden.
